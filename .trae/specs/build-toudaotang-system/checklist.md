# 验收检查清单

### M1: 项目基础设施
- [x] Next.js 项目可通过 `npm run dev` 正常启动
- [x] shadcn/ui 组件库已安装并可正常使用
- [x] 数据库 Schema 包含所有 15 张核心表，migration 可正常执行
- [x] Seed 脚本可填充示例数据（至少包含 1 个门店、3 个员工、5 个服务项目、10 个客户）
- [x] 登录页面可正常显示，使用手机号+密码可成功登录

- [x] 登录后根据角色正确跳转到对应端（管理端/员工端/用户端）

### M2: 门店与员工管理

- [x] 员工列表支持搜索

### M3: 服务项目管理
- [x] 可创建服务项目（名称、价格、时长、分类）

### M4: 客户管理
- [x] 可添加客户（姓名、手机号）
- [x] 可通过手机号搜索客户
- [x] 客户详情页展示余额、次卡、优惠券、订单记录
